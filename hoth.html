Rebel-hanger


Artifact:

The third artifact that I am choosing is from the course ACIT 2910. This is a team project to build a mock-up website for a restaurant on planet Hoth. The website is built using HTML and node.js and the test automation is done using selenium server with Nightwatch.

Go to » Rebel-hanger

Reflection:

This artifact links to the goals “Web Application development”, “UI/UX design”, “Software Design”, “database management” as well as the goal “Node programming”.


This project gave us the opportunity to work effectively as a team player and experience a full product development cycle from initial planning phase, to implementation phase, testing phase, and finally producing the deliverables.

The test automation with selenium server and nightwatch was totally new to me and they are important tools for sanity and functional testing for each and every iteration of the product development cycle. I have spent a long time to learn through online resources on setting up the environment for the test framework and learned the nightwatch APIs gradually to produce effective testing codes. Once the selenium server is setup with Travis-CI and linked with GitHub, it is so impressive how efficient and effective these tests are ran during every commit and Git push.


My goal was to get experience with software design and specifically with node.js programming. And also learn to setup selenium server to run nightwatch test automation with Travis-CI integration. And last but not the least, gain important teamwork experience for a full product development cycle.
