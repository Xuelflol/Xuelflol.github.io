Pig App


Artifact:

The second artifact that I am choosing is from the course ACIT 2515. I have created an application that can analyze Wireshark pcap data files and produce reports and graphs that can be used to analyze the network activities which can be useful for IT personals.

Go to » PigApp

Reflection:

This artifact links to the goals “UI/UX design”, “Software Design” as well as the goal “Java programming”.


This project shows that I can master the front-end JavaFx coding as well as the back-end data handling using java pcap libraries. It also showcase my use of Model-View-Controller design pattern gracefully and effectively.

I have some experience with Java programming from before and I really like the object oriented aspect of the language which makes my code modular and reusable. It was the first time for me to use JavaFx and Scene builder so I spent a lot of time viewing online materials for demos and code reviews. It was a group project and we named it piggy app with customized piggy theme. We have used table, pie chart and bar chart for front end and we made multiple scenes to represent different aspect of the data for the users. We had some issues with updating the table data in real time and pushing it back into the database. It was because we are not familiar with the pcap library and were not handling the data properly for conversion. We solved the issue by writing customized wrappers for the pcap lib to achieve our own goals.


Through this project, I have learned how to use third party libraries and how to extend the libraries for my own customization. I have also learned how to use JavaFx for front end development with Java applications. And I have also gained teamwork experience, and learned to allocate work and manage project effectively.
