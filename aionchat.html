Aion Chat


Artifact:

The first artifact is from the course ACIT2520. I have created an online chatroom application using node.js which is one of the most commonly used JS language in the industry for coding interactive server-client applications. I have deployed the application in Heroku.

Go to » Aion Chat Aion Chat Respository

Reflection:

This artifact links to the goals “Web Application development”, “UI/UX design”, “Software Design”, “database management” as well as the goal “Node programming”.


This application show case my strength in web application design especially in how to handle server-client interactions. The application uses sessions, jQuery and also database management technics. It also shows my ability to design UI/UX efficiently.

The project was to implement an interactive online chatroom app. It was the first time that I am working with node.js and my Java-script experience was quite minimal, so I had a lot to catch up at the beginning of the course.


I took on the project by myself and have decided to design forum for one of my favorite MMORPG game. I have collected a lot of relevant materials and data for front end and I have browsed many popular gaming forums to compare the features and layout in order to choose what I like and what I don’t. Then I have built the application back-end using what we were taught in class as well as information that I gathered along the way. During the implementation phase, I have encountered some issues for handling server and client data properly. I used a test driven approach for development so I was able to find the issues quite easily and I use multiple resources like “stack overflow” and GitHub to find solution for tricky issues. I have implemented all the features required in the project requirement and I have also finished all the bonus features that the instructors posted for us that were meant to be more of a challenge.


My goal was to get experience with software design and specifically with node.js programming. And also have more experience with UI/UX and database. Through this project, I have been able to achieve all these goals and connecting the dots together to present the final product in order to Ace the course.
